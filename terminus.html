GitHub - Nearly everything I've done is open source and free licence. The things that aren't, aren't important, aren't finished, or aren't mine.

HonkBonk - A Discord bot I made as an excuse to make some friends. It worked.

CDDL - GUI YouTube downloader. It was bad. Just use yt-dlp. I make command line programs now.

sprizzler - I got really in to PilotRedSky's music and art. This little filter gave me the confidence to do more art.

KitchComp - A companion program I made to help me with cooking. Calculates cost, a database of my available ingredients, and a tagged list of my recipes. It should have been a mobile app.

Discord archiver - Generates a website and database to look through old memories. I'll only discuss this with a therapist.

games

That Binds - A walking sim made at the darkest year of my life about suicide. Full of a time I don't want to share. They're in Assets/That Binds gdd.txt.

Photography - Inspired by my own hobby. A collectable gathering, stylized realism, import-your-3d-models furry game. I still like this idea a lot.

chek - Multiplayer chess game with custom boards, jank and modifiers. Made for a uni application. They ignored me.

hacky - A Fez inspired hacking game. I'd just recently made a couple languages with antlr3. Had ideas for fun puzzles, how each language was limited, etc. partly, I was working on it for my crush who loved Fez and comp sci. Still like the idea.

Rover - Game about controlling a rover from the inside, programming every function. Inspired by Duskers, Outer Wilds, and the Opportunity rover. Stopped when I learned about Iron Lung.

are we still friends - I can't link this one because it doxxes me. Similar to That Binds but much earlier. Game about suicide, you would explore the room, and especially computer, of someone who has gone missing. Inspired by Return Of The Obra Dinn.

False Dream - This is my "big one". My default project since 2019. It's gone thorugh four major stages that could each be considered its own game. I'm not linking it because I still work on it. If you've heard me discuss volcanos, ice ages or hyper light drifter, it was this game.
